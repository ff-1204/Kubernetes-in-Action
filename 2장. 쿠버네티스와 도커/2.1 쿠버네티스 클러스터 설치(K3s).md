# 1. 쿠버네티스 클러스터 설치(K3s)

## 1. WSL2 환경 준비

먼저 PowerShell(관리자 권한)에서 WSL2와 Ubuntu가 설치되어 있는지 확인합니다.

```powershell
wsl --list --verbose
```

Ubuntu가 없다면 설치합니다.

```powershell
wsl --install -d Ubuntu
```

설치 후 WSL 버전이 2인지 확인하세요. VERSION이 1이라면 다음 명령으로 변환합니다.

```powershell
wsl --set-version Ubuntu 2
```

## 2. WSL2 내 사전 설정

Ubuntu에 접속(중요!!)

```bash
PS C:\windows\system32> wsl
moon@YSMOON:/mnt/c/windows/system32$
```

패키지를 업데이트하고 필수 도구를 설치합니다.

```bash
sudo apt update && sudo apt upgrade -y
sudo apt install -y curl
```

curl -sfL https://get.k3s.io | INSTALL_K3s_CHANNEL=v1.35 sh -

## 4. K3s 설치 확인

K3s는 자체 kubectl을 포함

```bash
moon@YSMOON:/mnt/c/windows/system32$ sudo kubectl get nodes
NAME     STATUS   ROLES           AGE    VERSION
ysmoon   Ready    control-plane   3m3s   v1.35.3+k3s1
```

control-plane 노드가 `Ready` 상태로 나오면 정상입니다.

## 5. kubectl 편의 설정 (선택)

매번 `sudo kubectl`을 입력하기 번거로우니 일반 `kubectl`처럼 쓸 수 있게 설정합니다.

```bash
mkdir -p ~/.kube
sudo cp /etc/rancher/k3s/k3s.yaml ~/.kube/config
sudo chown $(id -u):$(id -g) ~/.kube/config
echo 'export KUBECONFIG=~/.kube/config' >> ~/.bashrc
source ~/.bashrc
```

이제 `kubectl get nodes`만으로 클러스터 조회가 가능합니다.

```
moon@YSMOON:/mnt/c/windows/system32$ kubectl get nodes
NAME     STATUS   ROLES           AGE     VERSION
ysmoon   Ready    control-plane   5m56s   v1.35.3+k3s1
```
